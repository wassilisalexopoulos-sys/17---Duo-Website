## Image Optimization Tips

### File Formats
- Use **JPEG** for photographs
- Use **WebP** for better compression (optional)
- Keep file sizes under 500KB each

### Color Treatment
- Convert to **black & white** or **desaturated** to match website aesthetic
- Maintain high contrast for impact
- Ensure DUO branding is visible

### Naming Convention
- Use descriptive, lowercase names with hyphens
- Include relevant keywords for SEO
- Keep names consistent with website references

## Upload Instructions

1. Save your images in the `images/` folder
2. Use the exact filenames listed above
3. Optimize images for web (compress, resize)
4. Test that all images load properly on the website

## Alternative Images

If you have different images that work better, you can:
1. Replace the filenames in the HTML files
2. Update the alt text to match your images
3. Ensure the images maintain the professional, coffee-focused theme

## Black & White Conversion

For the minimalist aesthetic:
- Convert color images to grayscale
- Increase contrast slightly for better impact
- Ensure DUO branding remains visible
- Test on both light and dark backgrounds
