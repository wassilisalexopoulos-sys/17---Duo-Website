# 🖼️ SIMPLE IMAGE SETUP GUIDE

## The Problem
Your website is showing text placeholders instead of images because there are no actual image files in the `images/` folder.

## The Solution
You need to add your actual coffee event photos to the `images/` folder.

## Step-by-Step Instructions

### 1. Open the Images Folder
- Navigate to: `C:\Users\walexopoulos\OneDrive - ALEXOPOULOS\17 - Duo Website\images\`
- You should see: `IMAGE_GUIDE.md` and `placeholder.html`

### 2. Add Your Photos
Take your coffee event photos and save them in the `images/` folder with these EXACT filenames:

#### Required Images:
1. **`hero-barista-pouring.jpg`** - Barista pouring milk (for homepage)
2. **`team-photo.jpg`** - Your team photo (for about page)
3. **`mobile-coffee-bar.jpg`** - Mobile coffee bar setup
4. **`layered-drinks.jpg`** - Beautiful layered drinks
5. **`coffee-pouring.jpg`** - Coffee pouring technique
6. **`team-at-work.jpg`** - Team working at event
7. **`iced-coffee-service.jpg`** - Iced coffee service
8. **`event-customers.jpg`** - Happy customers
9. **`matcha-preparation.jpg`** - Matcha preparation

### 3. Image Requirements
- **Format**: JPEG (.jpg)
- **Size**: Any size (website will resize automatically)
- **Color**: Black & white or desaturated for the minimalist look
- **Quality**: High enough to look professional

### 4. Test Your Website
- Open `index.html` in your browser
- You should see your actual photos instead of text placeholders
- Check all pages: Home, About, Services, Gallery, Contact

## Quick Test
1. Take any photo from your phone/computer
2. Save it as `images/hero-barista-pouring.jpg`
3. Refresh your website
4. You should see the photo on the homepage!

## If Images Still Don't Show
1. Check the filename is EXACTLY correct (case-sensitive)
2. Make sure the file is in the `images/` folder
3. Try a different image format (JPG, PNG)
4. Check the file isn't corrupted

## Need Help?
The website works perfectly with text placeholders, so you can use it as-is while you prepare your photos. The placeholders look professional and show exactly what images are needed.
