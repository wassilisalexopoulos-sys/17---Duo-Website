# DUO Coffee Events Website

A modern, minimalist website for a coffee event and catering company built with clean black & white aesthetics and strong typography.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern Typography**: Using Inter and Playfair Display fonts
- **Minimalist Aesthetic**: Clean black & white design with strategic use of grays
- **Smooth Animations**: Subtle scroll effects and hover animations
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Accessibility**: Focus states and keyboard navigation support

## Pages

1. **Homepage** (`index.html`) - Hero section, services preview, testimonials
2. **Services** (`services.html`) - Detailed service offerings with pricing
3. **About** (`about.html`) - Company story, team, and values
4. **Gallery** (`gallery.html`) - Portfolio of events and testimonials
5. **Contact** (`contact.html`) - Contact form and business information

## Design Guidelines

- **Color Scheme**: Black (#000000), White (#FFFFFF), Warm Gray (#F5F5F5)
- **Typography**: Playfair Display for headings, Inter for body text
- **Layout**: Grid-based responsive design with clean spacing
- **Images**: Placeholder areas ready for black & white photography

## Setup

1. Open `index.html` in a web browser
2. All pages are linked and functional
3. Images can be added to replace placeholder areas
4. Contact form is ready for backend integration

## Customization

- Replace placeholder images with actual photography
- Update contact information in footer sections
- Modify color scheme in CSS variables
- Add backend form processing for contact form
- Integrate with booking/calendar systems
